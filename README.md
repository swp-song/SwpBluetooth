### 导入

> * 手动导入：
>
>> * **SwpBluetooth** 文件夹导入项目中。
>> * **`#import "SwpBluetoothHeader.h`**
>>
> -------

> * CocoaPods 导入:
>
>> * **pod search SwpBluetooth**
>> * **pod 'SwpBluetooth'**
>> * **`#import <SwpBluetooth/SwpBluetoothHeader.h>`**
>>
> -------
